## Description

Clocker for [Keka](https://www.keka.com/).

## Features

- Endpoints for __Clocking-In__ and __Clocking-out__
- OTP and Captcha handling

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
